## Data science Docker container

Docker container built with Alpine linux containing a number of popular data science Python tools.

### Miniconda Python
* Pandas
* Matplotlib
* Seaborn
* Altair
* scikit-learn 
* lightgbm 
* xgboost 
* keras

### Unix tools
* oh-my-zsh
* vim
* git
